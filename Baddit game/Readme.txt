What is OverTheWire Bandit?

    OverTheWire Bandit is an online game that helps you learn how to use the Linux command line. You'll connect to a server, solve puzzles by typing commands, and unlock passwords to advance to the next level.
How to Play?

    The game is hosted on a remote server. You will connect to that server and solve puzzles using Linux commands.
    Each level has a password hidden somewhere on the server. Once you find it, you use it to unlock the next level.
